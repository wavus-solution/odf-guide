## API
```
Style(param, originalOptions): void
getTarget(): Feature
setTarget(target): void
modify(optDir, val): Style
getJSON(): String
getObject(): Object
produceElement(size, labelFlag): HTMLElement
```

## Style

스타일 객체 생성  

### new Style(param, originalOptions)

| Param | Type | Description |
| --- | --- | --- |
| param | `Object` | ol 스타일 생성을 위한 option |
| originalOptions | `odf_styleOption` | ODF 스타일 생성을 위한 option |

**Example**  
```js
let st = odf.StyleFactory.produce({
          image : {
              circle : {
                  radius:50,//크기
                  fill:{
                    color:'gray'//채우기 색
                  },//채우기
                  stroke: {//윤곽선
                      color:'red',//테두리 색
                      width:10,//굵기
                      lineDash:[4, 8]//점선 설정
                  },
              }
          },
      });//Style 객체
```

### getTarget() → Feature \| `Layer` \| `ExtDraw`
스타일이 입힌 대상을 반환

**Returns**: `Feature` \| `Layer` \| `ExtDraw` - 스타일이 입힌 대상  
**Example**  
```js
let st = odf.StyleFactory.produce({... });//Style 객체
 st.setTarget(feature);//feature에 스타일을 입힘
 st.getTarget(); //setTarget으로 셋팅한 스타일을 입힌 대상 조회(feature) return
```

### setTarget(target)
스타일을 입힐 대상을 셋팅

| Param | Type | Description |
| --- | --- | --- |
| target | `Feature` \| `Layer` \| `ExtDraw` | 스타일을 입힐 수 있는 대상 |

**Example**  
```js
let st = odf.StyleFactory.produce({... });//Style 객체
      st.setTarget(feature);//feature에 스타일을 입힘
```

### modify(optDir, val) → Style(#Style)
생성된 스타일 옵션을 변경하여 스타일을 새로 생성하여 반환

→ Style: 변경된 스타일 옵션을 적용한 스타일 객체  

| Param | Type | Description |
| --- | --- | --- |
| optDir | `String` | 수정할 옵션의 위치 (ex)image.radius |
| val | `Object` | 변경할 값 (ex)40 |

**Example**  
```js
let st = odf.StyleFactory.produce({... });//Style 객체
    st.modify('image.circle.radius',40);//radius 값 변경
```

### getJSON() → String
생성된 ODF 스타일 객체를 'ODF스타일 생성을 위한 JSON String 타입'으로 반환
```javascript
    let st = odf.StyleFactory.produce({...});//Style 객체
    st.getJSON();
```

→ String: ODF Style 생성을 위한 Option의 json string 형태  

### getObject() → Object
생성된 ODF 스타일 객체를 'ODF스타일 생성을 위한 Object 타입'으로 반환

→ Object: ODF Style 생성을 위한 Option  

### produceElement(size, labelFlag) → HTMLElement
스타일 옵션으로 범례용 element 생성

→ HTMLElement: 해당 스타일을 표현하는 htmlElement  

| Param | Type | Description |
| --- | --- | --- |
| size | `Number` | element 크기(width,height등에 사용). default 10 최소 0, 최대 2000 |
| labelFlag | `Boolean` | 라벨 스타일 적용 여부. true=&gt; 적용, false=&gt;미적용. default false |
