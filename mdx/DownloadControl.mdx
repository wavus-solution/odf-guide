## API
```
DownloadControl(options): void
getConstructorOptions(): Array
setMap(_map, createElementFlag): void
removeMap(): void
downloadsImage(): void
getBase64(): void
downloadsPDF(): void
```

## DownloadControl

DownloadControl 생성자

```javascript
 let downloadControl = new odf.DownloadControl({});
```  

### new DownloadControl(options)

| Param | Type | Description |
| --- | --- | --- |
| options | `Object` | PrintControl 객체생성 옵션 |
| options.targetElementSelector | `HTMLElement` | 프린트 대상 element 선택자 |
| options.exceptionElementSelectors | `Array.<String>` | 프린트 제외 element 선택자 |
| options.proxyURL | `String` | 프록시 url |
| options.proxyParam | `String` | 프록시 매개변수 명 |

### getConstructorOptions() → Array.<Object>
컨트롤 생성 옵션 반환

→ Array.<Object>: 컨트롤 생성 옵션 반환  

### setMap(_map, createElementFlag)
DownloadControl map 객체 연결

| Param | Type | Description |
| --- | --- | --- |
| _map | `Map` | DownloadControl 객체와 연결할 Map 객체 |
| createElementFlag | `Boolean` | DownloadControl 버튼 생성 여부 : default = true (Element) 생성, false = (Element) 생성 x |

### removeMap()
DownloadControl map 객체 연결 해제

### downloadsImage()
DownloadControl  이미지 다운로드 함수

### getBase64()
DownloadControl  지도 이미지를 base64로 반환

### downloadsPDF()
DownloadControl PDF 다운로드 함수
