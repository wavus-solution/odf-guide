## API
```
Marker(options): void
getElement(): Element
getPosition(): Coordinate
setPosition(coord): void
setPositioning(position): void
setOffset(position): void
getMap(): Map
setMap(Map): void
setDraggable(draggable): void
getDraggable(): Boolean
removeMap(): void
```

## Marker

마커 생성자

 ```javascript
//기본 마커 생성
let marker = new odf.Marker({
  position : new odf.Coordinate(955156.7761, 1951925.0984),
  //position : [955156.7761, 1951925.0984],
 });
 marker.setMap(map);
```

 
   

### new Marker(options)

| Param | Type | Description |
| --- | --- | --- |
| options | [`MarkerOption`](#MarkerOption) | 마커 생성을 위한 option |

### getElement() → Element
마커와 연결된 element 조회

→ Element: 마커와 연결된 Html element  

### getPosition() → Coordinate
마커의 위치정보 반환

→ Coordinate: 마커의 위치정보  

### setPosition(coord)
마커의 위치정보 변경

| Param | Type | Description |
| --- | --- | --- |
| coord | `Coordinate` \| `Array.<Number>` | 마커의 위치정보 |

### setPositioning(position)
마커의 속성과 관련하여 실제로 배치되는 방식을 정의

| Param | Type | Description |
| --- | --- | --- |
| position | `String` | 지도의 해당 지점을 기준으로 배치되는 방식 |

### setOffset(position)
배치할 때 사용되는 오프셋(픽셀)입니다. 배열의 첫 번째 요소는 수평 오프셋(양수 값은 오버레이를 오른쪽으로 이동) 배열의 두 번째 요소는 수직 오프셋(양수 값은 오버레이를 아래로 이동합니다.)

| Param | Type | Description |
| --- | --- | --- |
| position | `String` | 지도의 해당 지점을 기준으로 배치되는 방식 |

### getMap() → Map
마커에 연결된 map객체 반환

→ Map: 마커에 연결된 map객체, 없으면 undefined  

### setMap(Map)
마커에 map객체 연결

| Param | Type | Description |
| --- | --- | --- |
| Map | `Map` | 마커에 연결할 map객체 |

### setDraggable(draggable)
마커의 드래그 가능 여부 변경

| Param | Type | Description |
| --- | --- | --- |
| draggable | `Boolean` | 마커의 드래그 가능 여부 |

### getDraggable() → Boolean
마커의 드래그 가능 여부 반환

→ Boolean: 마커의 드래그 가능 여부  

### removeMap()
마커와 연결된 map 객체 제거

## MarkerOption
**Type**: `Object`
**Properties**

| Name | Type | Description |
| --- | --- | --- |
| position | `Coordinate` \| `Array.<Number>` | 마커의 위치 |
| draggable | `Boolean` | 드래그 가능 여부(기본값 false) |
| positioning | [`odf_marker_positioning`](#odf_marker_positioning) | 마커의 상대적 위치 (기본값 'bottom-center') |
| offset | `Array.<number>` | 기준점으로 부터 정의한 값만큼 마커를 이동(픽셀) (기본값 [0,0])  - 첫번째 요소 : 수평 오프셋 값. 양수는 마커를 오른쪽으로 이동시킴  - 두번째 요소 : 수직 오프셋 값. 양수는 마커를 아래로 이동시킴 |
| stopEvent | `Boolean` | 마커 이벤트 전파 중지 여부. true =&gt; 이벤트 전파 중지, false=&gt; 이벤트 전파 (기본값 false) |
| style | `Object` | 마커의 스타일 정의 |
| style.width | `String` | 마커의 너비 정의 |
| style.height | `String` | 마커의 높이 정의 |
| style.src | `String` | 마커의 배경 이미지 url 정의 |
| style.element | `HTMLElement` | 사용자 정의 마커 element |
| autoPan | `Boolean` | 지도 화면에서 팝업이 잘려나오게 될 경우, 지도를 이동할지 여부 - true : 지도 화면에서 팝업이 잘려나오게 될 경우, 지도를 이동합니다. (기본값 : false) |
| autoPanAnimation | `number` | autoPan이 ture일때, 지도를 이동하는데 사용되는 애니메이션 입니다. (기본값 : 250) 최소 :0 최대 : 100000 |
| autoPanMargin | `number` | autoPan이 ture일때, 팝업과 지도 사이의 여백(픽셀) 지정 (기본값 : 20) |
