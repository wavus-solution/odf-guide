## API
```
addListener(target, eventType, callback, expiration): String
removeListener(evtneId): void
trigger(target, eventType, param): void
hasEventId(eventId): Boolean
hasEvent(target, eventType): number
stateChange(eventId, state, excludeThis): boolean
excludeEvent(parameter): Array
```

## event

### addListener(target, eventType, callback, expiration) → String
이벤트 등록

```javascript
//이벤트 연결
   let map = new odf.Map(document.getElementById('map'), {
  center: new odf.Coordinate(955673.3095379177, 1954893.4607722224),
  zoom: 17,
  projection: 'EPSG:5179',
  maxZoom: 20,
  minZoom: 8,
});
   //map 객체에 클릭 이벤트 연결
   let clickEventId = odf.event.addListener(map,'click',(evt)=>{
    //클릭한 위치에 마커 추가
  let marker = new odf.Marker({
    position : new odf.Coordinate(evt.coordinate)
  });
  marker.setMap(map);
});
```

→ String: 이벤트를 특정하는 Id  

| Param | Type | Default | Description |
| --- | --- | --- | --- |
| target | `String` \| `HTMLElement` \| `odf.ExtObject` |  | 이벤트를 연결할 대상(css 선택자 텍스트, odf.ExtObject) (ex)   '#moveBtn' | '.mtbBtn' | (new Map(&#123;...&#125;)) |
| eventType | [`event_type`](#event_type) |  | target에 연결 가능한 이벤트 종류 (ex) 'click', 'mousedown',   'moveend','custom-event-01' |
| callback | `function` |  | 이벤트가 발생 될때 실행할 function |
| expiration | [`event_expiration`](#event_expiration) | `odfDeleted` | event 해제 시기별 type. |

### removeListener(evtneId)
이벤트 해제

| Param | Type | Description |
| --- | --- | --- |
| evtneId | `String` | 이벤트를 특정하는 Id |

### trigger(target, eventType, param)
이벤트 강제로 실행

| Param | Type | Description |
| --- | --- | --- |
| target | `String` \| `HTMLElement` \| `odf.ExtObject` \| `MapBase` | - 강제로 이벤트를 실행시킬 대상(css 선택자 텍스트, odf.ExtObject)         (ex) '#moveBtn' | '.mtbBtn' | (new Map(&#123;...&#125;)) |
| eventType | `String` | 강제로 실행시킬 이벤트의 종류         (ex) 'click', 'mousedown', 'moveend','custom-event-01' |
| param | `Object` | 이벤트를 강제 실행시키면서 해당 콜백 function에 넘길 파라미터 |

### hasEventId(eventId) → Boolean
특정 id를 갖는 이벤트가 등록되어있는지 확인

→ Boolean: - 이벤트 등록 여부 반환  

| Param | Type | Description |
| --- | --- | --- |
| eventId | `String` | 이벤트 등록 여부를 확인할 eventId |

### hasEvent(target, eventType) → number
특정 객체에 특정 타입의 이벤트가 있는지 확인

→ number: 이벤트 타겟에 등록된 이벤트 갯수  

| Param | Type | Description |
| --- | --- | --- |
| target | `String` \| `HTMLElement` \| `odf.ExtObject` | 이벤트 타겟 |
| eventType | [`event_type`](#event_type) | target에 연결 가능한 이벤트 종류         (ex) 'click', 'mousedown', 'moveend','custom-event-01' |

### stateChange(eventId, state, excludeThis) → boolean
queue에 등록된 event의 상태값 변경

→ boolean: - 작업 성공 여부 (ex) true=&gt; 성공, false =&gt;실패  

| Param | Type | Description |
| --- | --- | --- |
| eventId | `string` | 상태를 변경할 이벤트의 id |
| state | `event_state` | 이벤트 상태 |
| excludeThis | `boolean` | 입력한 eventId를 제외하고 eventId에 해당하는 이벤트 target과 이벤트 type이 일치하는 다른 이벤트들을 대상으로 할지 여부 - true : 입력한 eventId를 제외하고 eventId에 해당하는 이벤트 target과 이벤트 type이 일치하는 다른 이벤트들을 대상으로 처리 - false : (기본값)입력한 eventId를 대상으로 처리 |

### excludeEvent(parameter) → Array.<event_excludeEvent_item>(#event_excludeEvent_item)
queue에 등록된 event의 상태값 변경

→ Array.<event_excludeEvent_item>: 대상 이벤트 id 배열  

| Param | Type | Description |
| --- | --- | --- |
| parameter | `Object` | 대상 이벤트 |
| parameter.eventId | `string` | 이벤트 id |
| parameter.target | `String` \| `HTMLElement` \| `odf.ExtObject` | 이벤트 타겟 |
| parameter.eventType | [`event_type`](#event_type) | 이벤트 타입 |

## Type Definitions

## event_expiration : `Boolean` \| `String`
**Properties**

| Name | Type | Description |
| --- | --- | --- |
| true | `Boolean` | 한번만 호출하고 해제 |
| false | `Boolean` | 지도객체가 없어지거나 disconncet 할 때 해제 |
| customize | `String` | 사용자가 원하는 시점에서 1번 호출하고 해제 |
| odfDeleted | `String` | (기본값) 지도객체가 없어지면 해제 |
