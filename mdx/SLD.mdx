## API
```
SLD(options): void
getSLD(): Promise
setLayer(wmslayer): void
getObject(): odf_sld_option
getJSON(): String
download(): void
produceElement(size, labelFlag): HTMLElement
```

## SLD

SLD 생성자  

### new SLD(options)

| Param | Type | Description |
| --- | --- | --- |
| options | `odf_sld_option` | SLD 생성 옵션 |

**Example**  
```javascript
   let sld = odf.StyleFactory.produceSLD({
	   rules: [
		   {
			   name: 'My Rule', //룰 이름
         symbolizers: [
           {
             kind: 'Mark',
             // 포인트에 표현될 도형 종류
             // - 'circle' : 원
             // - 'square' : 사각형
             // - 'triangle' : 세모
             // - 'star' : 별 모양
             // - 'cross' :  + 모양
             // - 'x' : x 모양
             wellKnownName: 'Square',
  
             //포인트에 표현될 도형의 반지름
             radius: 6,
             //포인트에 표현될 도형의 채우기색
             color: '#FF0000',
             //포인트에 표현될 도형의 채우기색 투명도 0~1
             fillOpacity: 0.5,
             //포인트에 표현될 도형의 윤곽선 색
             strokeColor: '#0000FF',
             //포인트에 표현될 도형의 윤곽선 투명도 0~1
             strokeOpacity: 0.7,
             //포인트에 표현될 도형의 윤곽선 두께
             strokeWidth: 3,
           },
         ],
       },
     ]
   });
```

### getSLD() → Promise
sld 형식의 문자열 조회

→ Promise: 생성된 sld 형식의 문자열이 promise value로 담긴 promis 객체  

### setLayer(wmslayer)
해당 SLD를 WMS에 적용

| Param | Type | Description |
| --- | --- | --- |
| wmslayer | `Layer` | 해당 SLD를 적용할 WMS 레이어 |

### getObject() → odf_sld_option
SLD 생성 옵션 조회

→ odf_sld_option: SLD 생성 옵션  

### getJSON() → String
SLD 생성 옵션을 JSON 타입으로 변환

→ String: JSON 타입으로 변환된 SLD 생성 옵션  

### download()
SLD 파일 다운로드

### produceElement(size, labelFlag) → HTMLElement
SLD 스타일에 맞는 HTMLElement 생성

→ HTMLElement: SLD 스타일에 맞게 생성된 HTMLElement  

| Param | Type | Description |
| --- | --- | --- |
| size | `Numbe` | HTMLElement 크기 최소 0 ~ 최대 2000 |
| labelFlag | `Boolean` | 라벨 표현 여부 |
