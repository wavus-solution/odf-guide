## API
```
getConstructorOptions(): Array
setMap(_map, createElementFlag): void
removeMap(): void
clear(): void
```

## ClearControl

ClearControl 생성자

 ```javascript
 let clearControl = new odf.ClearControl();//ClearControl 생성
```  

### getConstructorOptions() → Array.<Object>
컨트롤 생성 옵션 반환

→ Array.<Object>: 컨트롤 생성 옵션 반환  

### setMap(_map, createElementFlag)
ClearControl map 객체 연결
clear HTMLElement를 만들고, 이벤트 바인딩 시켜, map 객체와 연결

 

| Param | Type | Description |
| --- | --- | --- |
| _map | `Map` | ClearControl 객체와 연결할 Map 객체 |
| createElementFlag | `Boolean` | ClearControl 버튼 생성 여부 : default = true (Element) 생성, false = (Element) 생성 x |

### removeMap()
ClearControl map 객체 연결 해제

### clear()
ClearControl map 객체에서 그리기 이벤트 삭제
