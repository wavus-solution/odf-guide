## DrawControl

지도에 그릴수 있는 다양한 그리기 도구를 제공

```javascript
let drawControl = new odf.DrawControl({
        //id : 'myDrawControl' //drawControl 고유 ID (createNewLayer: true 시 자동 생성됨),
        //style : {} //그리기 피쳐 odf 스타일 옵션,
        //bufferStyle : {} //버퍼 도형 odf 스타일 옵션,
        //createNewLayer : true //drawControl 생성 시 새로운 레이어 생성 여부
        //message : { //그리기 도구 시작시 툴팁 메세지
          //DRAWSATRT_POINT : '점 그리기 측정입니다.',
          //DRAWSTART_LINESTRING : '',
          //DRAWSTART_POLYGON : '',
          //DRAWSTART_CURVE : '' ,
          //DRAWSTART_TEXT : '',
          //DRAWSTART_BUFFER : '',
          //DRAWSTART_CIRCLE : '',
          //DRAWSTART_BOX : '',
          //DRAWEND_DRAG : '',
          //DRAWEND_DBCLICK : '',
        //},
        //measure : false, 측정 옵션 활성화 여부, (선, 원형만 측정 가능)
        //continuity : false, 연속 측정 여부  (기본값 false),
        //rightClickDelete : false, 우클릭 삭제 기능 활성화 여부,
        //tools : ['text', 'polygon','lineString','point','circle','curve','box','square','buffer'],
});
```  

### new DrawControl([options])

| Param | Type | Description |
| --- | --- | --- |
| [options] | [`DrawControlOption`](#DrawControlOption) | DrawControl 객체생성 옵션 |
| options.id | `String` | drawControl 고유 ID 설정. 미지정 시 createNewLayer가 true면 자동 생성, false면 'draw' |
| options.style | `odf_styleOption` | 그리기 도형 style 설정 옵션 |
| options.bufferStyle | `odf_styleOption` | 버퍼 그리기 도형 style 설정 옵션 |
| options.createNewLayer | `boolean` | drawControl 생성 시 마다 새 레이어 생성 여부 |
| options.message | `Object` | 툴팁 메세지 |
| options.message.DRAWSTART_POINT | `String` | 점 그리기 시작 안내 메세지 (기본값) '점을 그리기 위해 지도를 클릭해주세요' |
| options.message.DRAWSTART_LINESTRING | `String` | 선 그리기 시작 안내 메세지 (기본값) '선을 그리기 위해 지도를 클릭해주세요' |
| options.message.DRAWSTART_POLYGON | `String` | 면 그리기 시작 안내 메세지 (기본값) '면을 그리기 위해 지도를 클릭해주세요' |
| options.message.DRAWSTART_CURVE | `String` | 곡선 그리기 시작 안내 메세지 (기본값) '곡선을 그리기 위해 지도를 드래그해주세요' |
| options.message.DRAWSTART_TEXT | `String` | 텍스트 그리기 시작 안내 메세지 (기본값) '텍스트를 입력하기 위해 지도를 클릭해주세요.' |
| options.message.DRAWSTART_BUFFER | `String` | 버퍼 그리기 시작 안내 메세지 (기본값) '버퍼를 생성하기 위해 레이어를 선택해주세요.' |
| options.message.DRAWSTART_CIRCLE | `String` | 원 그리기 시작 안내 메세지 (기본값) '원을 그리기 위해 지도를 클릭해주세요.' |
| options.message.DRAWSTART_BOX | `String` | 사각형 그리기 시작 안내 메세지 (기본값) '사각형을 그리기 위해 지도를 클릭해주세요.' |
| options.message.DRAWEND_DRAG | `String` | 그리기 종료 안내 메세지(드래그) (기본값) '드래그를 멈추면 그리기가 종료됩니다.' |
| options.message.DRAWEND_DBCLICK | `String` | 그리기 종료 안내 메세지(더블클릭) (기본값) '더블클릭시 그리기가 종료됩니다.' |
| options.measure | `Boolean` | 측정 옵션 활성화 여부(기본값 false) |
| options.continuity | `Boolean` | 연속 그리기 여부 (기본값 false) |
| options.rightClickDelete | `Boolean` | 우클릭 삭제 기능 활성화 여부 (기본값 false) |
| options.maxAreaSize | `Number` | 그리기 도형 면적 제한 (1000000 = 1㎢). box, circle, square 그리기 중 도형 크기가 maxAreaSize를 초과하지 않도록 제한 |
| options.tools | `Array.<String>` | 생성할 툴 배열 |
| options.tools.text | `String` | 텍스트 그리기 툴 |
| options.tools.polygon | `String` | 다각형 그리기 툴 |
| options.tools.lineString | `String` | 선 그리기 툴 |
| options.tools.box | `String` | 사각형 그리기 툴 |
| options.tools.square | `String` | 정사각형 그리기 툴 |
| options.tools.point | `String` | 점 그리기 툴 |
| options.tools.circle | `String` | 원 그리기 툴 |
| options.tools.curve | `String` | 곡선 그리기 툴 |
| options.tools.buffer | `String` | 버퍼 그리기 툴 |

## DrawControlOption
**Type**: `Object`
DrawControl 생성 옵션 타입

**Properties**

| Name | Type | Description |
| --- | --- | --- |
| [id] | `string` | drawControl 고유 ID |
| [style] | `Object` | 그리기 도형 스타일 옵션 |
| [bufferStyle] | `Object` | 버퍼 도형 스타일 옵션 |
| [createNewLayer] | `boolean` | 생성 시 새 레이어 생성 여부 |
| [message] | `Object` | 툴팁 메세지 옵션 |
| [measure] | `boolean` | 측정 옵션 활성화 여부 (기본값 false) |
| [continuity] | `boolean` | 연속 그리기 여부 (기본값 false) |
| [rightClickDelete] | `boolean` | 우클릭 삭제 기능 활성화 여부 (기본값 false) |
| [maxAreaSize] | `number` | 그리기 도형 면적 제한 (1000000 = 1㎢) |
| [tools] | [`Array.<DrawType>`](#DrawType) | 생성할 툴 배열 |
