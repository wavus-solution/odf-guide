## API
```
HomeControl(options): void
setMap(_map, createElementFlag): void
removeMap(): void
moveHome(): void
changeOption(options): void
```

## HomeControl

HomeControl 생성자

 ```javascript
 let homeControl = new odf.HomeControl(options);//HomeControl 생성
```  

### new HomeControl(options)

| Param | Type | Description |
| --- | --- | --- |
| options | `Object` | homeControl 생성 옵션 //입력 없을 시 맵 옵션 초기값 |
| options.zoom | `Number` | 줌 레벨 값 |
| options.resolution | `Number` | 레졸루션 값 (zoom 보다 우선적용) |
| options.center | `Coordinate` | 중심점 좌표 |
| options.projection | `String` | 중심점 좌표계 정보(EPSG:XXXX 형식) |

### setMap(_map, createElementFlag)
HomeControl map 객체 연결
clear HTMLElement를 만들고, 이벤트 바인딩 시켜, map 객체와 연결

 

| Param | Type | Description |
| --- | --- | --- |
| _map | `Map` | HomeControl 객체와 연결할 Map 객체 |
| createElementFlag | `Boolean` | HomeControl 버튼 생성 여부 : default = true (Element) 생성, false = (Element) 생성 x |

### removeMap()
HomeControl map 객체 연결 해제

### moveHome()
설정 위치 홈으로 이동

### changeOption(options)
옵션 변경

| Param | Type | Description |
| --- | --- | --- |
| options | `Object` | 설정을 변경할 obeject |
| options.center | `Array` | 변경할 지도 중심좌표 |
| options.zoom | `Number` | 변경할 지도 확대레벨 (예제 코드 참조) |
