## Map

Map 생성 클래스
   ```javascript
   let mapContainer = document.getElementById('map');
   const center = new odf.Coordinate(955156.7761, 1951925.0984);
   let mapOption = {
     center: center,
     zoom: 15,
     projection: 'EPSG:5179',
     maxZoom: 20,
     minZoom: 8,
   };
   let map = new odf.Map(mapContainer, mapOption);
   ```

### selectFeature(parameters) → Array.<Feature>
지도에서 클릭한 Feature를 선택

→ Array.<Feature>: 선택된 영역으로 부터 포함관계인 피쳐 배열  

| Param | Type | Default | Description |
| --- | --- | --- | --- |
| parameters | `Object` |  |  |
| [parameters.extractType] | `&#x27;draw&#x27;` \| `&#x27;view&#x27;` \| `&#x27;pixel&#x27;` \| `&#x27;feature&#x27;` \| `&#x27;cql&#x27;` |  | 피쳐 추출 유형 - draw 직접 그린 도형 내에 속한 피쳐 추출 - view 현재 지도영역에서 피쳐 추출 - pixel 특정 픽셀에 곂치는 피쳐 추출 - feature 특정 도형(compareFeature) 내에 속한 피쳐 추출 - cql cql_filter로 피처 추출 |
| [parameters.drawType] | `&#x27;polygon&#x27;` \| `&#x27;box&#x27;` \| `&#x27;circle&#x27;` \| `&#x27;point&#x27;` |  | 그리기 유형. extractType의 값이 'draw'일 경우 필수값 - polygon 다각형 - box 사각형 - circle 원형 - point 점 |
| [parameters.targetLayer] | `Layer` |  | 특정 Layer 내에서만 feature 추출, 미입력시 모든 레이어에서 추출 |
| [parameters.featureReturn] | `Boolean` |  | 필터링 대상이 되는 도형 또는 영역 정보를 함께 리턴할지 여부(기본값 false) |
| [parameters.pixel] | `Number` |  | 피쳐 추출을 위한 좌표 기준. extractType의 값이 'pixel'일 경우 필수값. |
| [parameters.callback] | `function` |  | 그리기 영역으로 부터 선택된 피쳐 배열을 리턴받을 콜백 함수. 없으면 selectFeature 함수에서 결과 반환됨(extractType의 값이 'draw' 타입일 경우 제외) |
| [parameters.permanentDrawLifeCycle] | `Boolean` |  | 영구적 그리기 interaction 활성화 여부. extractType의 값이 'draw'일 경우 해당(기본값 true) |
| [parameters.continuity] | `Boolean` |  | 연속 그리기 여부. extractType 값이 'draw'이고 permanentDrawLifeCycle 값이 true인 경우에만 사용 가능(기본값 false) |
| [parameters.compareFeature] | `ODF.Feature` |  | extractType이 feature일 경우, 이 도형 내에 속한 피처 추출 - point 로 geometryType 제한 |
| [parameters.includeNoGeometryData] | `Boolean` |  | geometry가 없는 feature도 조회할지 여부(기본값 false, geometry 없는 feature 조회  x) |
| [parameters.cql] | `String` |  | cql 문자열 |
| [parameters.maxAreaSize] | `Number` |  | 그리기 도형 크기 제한(1000000이 1㎢) |
| [parameters.restrictDrawSize] | `Boolean` |  | box, circle 그리기 중 도형 크기가 maxAreaSize를 초과하지 않도록 제한 (기본값 false). polygon은 그린 후 검증만 가능 |
| [parameters.pointBuffer] | `Number` |  | 대상 좌표에 버퍼를 지정 (단위:픽셀) (기본값 20) ※ 피처 추출 유형이 pixel이거나 draw이면서 drawType이 point일 경우 적용됨 |
| [parameters.message] | `Object` |  | extractType이 'draw'일때, message 값 정의시 툴팁메세지에 사용자 정의 메세지 적용 |
| parameters.message.DRAWSTART_POINT | `String` |  | 점 그리기 시작 안내 메세지 (기본값) '점을 그리기 위해 지도를 클릭해주세요' |
| parameters.message.DRAWSTART_POLYGON | `String` |  | 면 그리기 시작 안내 메세지 (기본값) '면을 그리기 위해 지도를 클릭해주세요' |
| parameters.message.DRAWSTART_CIRCLE | `String` |  | 원 그리기 시작 안내 메세지 (기본값) '원을 그리기 위해 지도를 클릭해주세요.' |
| parameters.message.DRAWSTART_BOX | `String` |  | 사각형 그리기 시작 안내 메세지 (기본값) '사각형을 그리기 위해 지도를 클릭해주세요.' |
| options.message.DRAWEND_DBCLICK | `String` |  | 그리기 종료 안내 메세지(더블클릭) (기본값) '더블클릭시 그리기가 종료됩니다.' |
| [parameters.useVisible] | `Boolean` |  | 레이어 visible 여부에 따라 selectFeature 대상 변경 |
| [parameters.polygonRightClickFinish] | `Boolean` | `false` | 다각형(polygon) 우클릭 종료 여부 (drawType이 'polygon'일 때만 적용) - false: 더블클릭으로 종료 (기본값) - true: 우클릭으로 종료 |
