## Feature

### new Feature([param])
Feature 생성자

| Param | Type | Description |
| --- | --- | --- |
| [param] | [`FeatureOption`](#FeatureOption) | Feature 생성 옵션. 미입력시 빈 Feature 생성 |

### changed()
개정 카운터를 늘리고 'change' 이벤트를 전달
```javascript
feature.changed()
```

### clone() → Feature(#Feature)
feature 객체 복사. 원본 feature에 geometry가있는 경우 geometry도  복제

→ Feature: 복사된 feature 객체  

### dispatchEvent(event)
이벤트를 전달하고이 유형의 이벤트를 수신하는 모든 리스너를 호출

| Param | Type | Description |
| --- | --- | --- |
| event | `event` | 전달할 이벤트 |

### getGeometry() → Geometry
해당 feature에 등록된 geometry를 조회

→ Geometry: - 해당 feature에 등록된 geometry  

### getGeometryName() → String
해당 feature에 등록된 geometry의 이름 조회

→ String: - 해당 feature에 등록된 geometry의 이름  

### get(key-)
해당 feature에 등록된 속성 값을 조회

| Param | Type | Description |
| --- | --- | --- |
| key- | `String` | 속성명 |

### set(key-, value-)
해당 feature에 속성 값 등록

| Param | Type | Description |
| --- | --- | --- |
| key- | `String` | 속성명 |
| value- | `String` | 속성 값 |

### getId() → String
해당 feature의 id 조회

→ String: id  

### getKeys() → Array.<String>
해당 feature 객체의 속성명 목록 조회

→ Array.<String>: 해당 feature 객체의 속성명 목록  

### getProperties() → Object
해당 feature 객체의 모든 속성의 이름과 값을 조회

→ Object: feature 객체의 모든 속성의 이름과 값  

### getRevision() → Number
해당 feature 객체의 버전 번호 조회

→ Number: 해당 feature 객체의 버전 번호  

### getStyleFunction() → function
해당 feature 객체에 등록된 스타일 function 조회

→ function: 해당 feature 객체에 등록된 스타일  

### setGeometry(geometry)
해당 feature 객체의 geometry 변경

| Param | Type | Description |
| --- | --- | --- |
| geometry | `Geometry` | 변경할 geometry |

### setGeometryName(geometryName)
해당 feature 객체의 geometry 이름 변경

| Param | Type | Description |
| --- | --- | --- |
| geometryName | `String` | 변경할 geometry 이름 |

### setId(id)
해당 feature 객체의 id 변경

| Param | Type | Description |
| --- | --- | --- |
| id | `String` | 변경할 geometry id |

### setProperties(properties)
해당 feature 객체의 속성 변경

| Param | Type | Description |
| --- | --- | --- |
| properties | `Object` | 변경할 속성 object |

### setStyle(properties)
해당 feature 객체의 속성 변경

| Param | Type | Description |
| --- | --- | --- |
| properties | `Object` | 변경할 속성 object |

### getStyle() → Style
해당 feature 객체에 등록된 스타일 조회

→ Style: 해당 feature 객체에 등록된 스타일  

### unset(key, opt_silent)
속성명 제거

| Param | Type | Description |
| --- | --- | --- |
| key | `String` | 제거할 속성명 |
| opt_silent | `Boolean` | 이벤트 전파 차단여부 |

### toPolygon(sides, angle) → Feature(#Feature)
geometry 타입이 circle인 feature를 polygon으로 변환

→ Feature: geometry 타입이 Polygon으로 변환된 피처  

| Param | Type | Description |
| --- | --- | --- |
| sides | `Number` | 꼭지점 갯수 Default : 32 |
| angle | `Number` | 첫번째 시작 각도(라디안) Default : 0 |

### polygonFromCircle(feature, sides, angle) → Feature(#Feature)
geometry 타입이 circle인 feature를 polygon으로 변환

→ Feature: geometry 타입이 Polygon으로 변환된 피처  

| Param | Type | Description |
| --- | --- | --- |
| feature | [`Feature`](#Feature) | 변환을 수행할 피처 |
| sides | `Number` | 꼭지점 갯수 Default : 32  최소 4, 최대 100 |
| angle | `Number` | 첫번째 시작 각도(라디안) Default : 0 최소 : 0, 최대 : 360 |

### buffer(meter) → Feature(#Feature)
해당 feature의 버퍼 도형을 만들어 반환

→ Feature: 버퍼 도형  

| Param | Type | Description |
| --- | --- | --- |
| meter | `Number` | 버퍼 크기 |

### setSource(source)
해당 feature를 source에 추가
 

| Param | Type | Description |
| --- | --- | --- |
| source | `Source` | 해당 feature를 추가할 대상 source |

### removeSource(source)
해당 feature를 source에서 제거
 

| Param | Type | Description |
| --- | --- | --- |
| source | `Source` | 해당 feature를 제거할 대상 source |

### setFeature(setFunction) → void
내부 feature 객체에 직접 접근하여 조작할 수 있는 함수

| Param | Type | Description |
| --- | --- | --- |
| setFunction | `function` | 내부 feature 객체를 인자로 받는 콜백 함수 |

### getArea() → number
해당 feature의 면적, 길이 제공
 

→ number: feature의 길이 또는 면적 측정값 (단위 : m)  

### toGeoJson(GeoJson)
해당 feature를 geojson 타입의 Object로 변환
 

| Param | Type | Description |
| --- | --- | --- |
| GeoJson | [`odf_feature_geojson_object`](#odf_feature_geojson_object) | 변환된 geojson 타입의 Object |

### featureToWKT([feature]) → Object
Feature를 WKT(Well-Known Text) 형식으로 변환

→ Object: WKT 문자열과 속성 정보를 담은 객체  

| Param | Type | Description |
| --- | --- | --- |
| [feature] | [`Feature`](#Feature) | 변환할 Feature 객체 (미입력시 현재 객체 사용) |

### featureFromWKT(wkt, [property]) → Feature(#Feature)
WKT(Well-Known Text) 문자열로부터 Feature 객체 생성

→ Feature: WKT로부터 생성된 Feature 객체  

| Param | Type | Default | Description |
| --- | --- | --- | --- |
| wkt | `string` |  | WKT 형식의 geometry 문자열 |
| [property] | `Object` | `` | Feature에 설정할 속성 객체 |

### getCenterPoint() → coordinates
피처의 중심점 반환
 

→ coordinates: 피쳐 중심점 좌표  

### hasProperties() → boolean
Feature가 속성을 가지고 있는지 여부 확인

→ boolean: 속성 보유 여부  

## odf_coordinate
**Type**: `Array.<number>`
좌표 배열 타입 [x, y]

## FeatureOption
**Type**: `Object`
Feature 생성 옵션 타입

**Properties**

| Name | Type | Description |
| --- | --- | --- |
| coordinates | `Coordinate` \| `Array.<number>` \| `Array.<Array.<number>>` \| `Array.<Array.<Array.<number>>>` \| `Array.<Array.<Array.<Array.<number>>>>` | 좌표 배열 |
| geometryType | `&#x27;point&#x27;` \| `&#x27;multipoint&#x27;` \| `&#x27;linestring&#x27;` \| `&#x27;multilinestring&#x27;` \| `&#x27;polygon&#x27;` \| `&#x27;multipolygon&#x27;` \| `&#x27;circle&#x27;` | geometry 타입 |
| [properties] | `Object` | 속성 데이터 |
| [style] | `odf_styleOption` | 스타일 옵션 |
| [circleSize] | `number` | circle 타입일 때 반지름 (미터 단위) |

## odf_feature_geojson_object
**Type**: `Object`

| Param | Type | Description |
| --- | --- | --- |
| geometry.type | [`odf_geometryType`](#odf_geometryType) | 지오메트리 타입 |
| geometry.coordinates | [`odf_coordinate`](#odf_coordinate) \| [`Array.<odf_coordinate>`](#odf_coordinate) \| `Array.<Array.<odf_coordinate>>` \| `Array.<Array.<Array.<odf_coordinate>>>` | 지오메트리   좌표값 |

**Properties**

| Name | Type | Description |
| --- | --- | --- |
| geometry | `Object` | 지오메트리 정보 |
