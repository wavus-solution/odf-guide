## Layer

### getInitialOption() → Object
LayerFactory를 통해 Layer 생성한 경우 , 생성 옵션 값을 반환
 ```javascript
let sample = odf.LayerFactory.produce(type, {
  server: 'GeoServer URL 입력',
  layer: 'Store 명칭:Layer 명칭 입력',
  service: '서비스 종류',
}); // sample은 Layer 클래스로 만들어진 객체
sample.getInitialOption();
```

→ Object: 레이어 생성 옵션

## LayerOption
**Type**: `Object`
레이어 생성 옵션 (LayerFactory.produce에서 사용)

**Properties**

| Name | Type | Description |
| --- | --- | --- |
| [server] | `string` \| `Object` | 서버 URL 또는 서버 옵션 객체 |
| [layer] | `string` | 레이어명 (저장소명:레이어명) |
| [service] | `string` | 서비스 타입 (wfs, wms, group 등) |
| [limit] | `number` | 호출 제한 수 |
| [crtfckey] | `string` | API 인증 키 |
| [bbox] | `boolean` | bbox 사용 여부 (wfs) |
| [method] | `&#x27;get&#x27;` \| `&#x27;post&#x27;` | HTTP 메소드 |
| [tiled] | `boolean` | 타일링 여부 (wms) |
| [geoJson] | `Object` | GeoJSON 데이터 |
| [style] | `Object` | 스타일 옵션 |
| [odfId] | `string` | 레이어 ID |
| [cqlFilter] | `string` | CQL 필터 |
| [projection] | `string` | 좌표계 |
| [opacity] | `number` | 투명도 |
| [visible] | `boolean` | 가시성 |
| [zIndex] | `number` | z-index |
| [minZoom] | `number` | 최소 줌 |
| [maxZoom] | `number` | 최대 줌 |
