## API
```
ZipControl(options): void
setMap(map): void
```

## ZipControl

ZipControl 생성자

 ```javascript
 let options = {
   file: document.getElementById('uploadBtn'),
   encoding: 'UTF-8',
   epsg: '5179',
 };
 let zipControl = new odf.ZipControl(options)//ZipControl 생성
```  

### new ZipControl(options)

| Param | Type | Description |
| --- | --- | --- |
| options | `Object` | ZipControl 객체생성 옵션 |
| options.file | `HTMLObjectElement` | zip 객체생성 옵션(File type input HTML) |
| options.encoding | `String` | Encoding 종류 |
| options.epsg | `String` | 좌표계 EPSG 종류(EPSG 문자를 제외한 코드 번호) |
| options.getLayer | `function` | 생성된 Layer 객체를 받을 callback  함수 |
| options.geometryType | `Array` | 레이어 생성 시 허용할 geometry 타입배열 ['point','linestring','polygon'] 만 허용 |
| options.errorCallback | `function` | 에러 발생 시 에러 메시지 리턴 받을 함수 |

### setMap(map)
ZipControl map 객체 연결

| Param | Type | Description |
| --- | --- | --- |
| map | `Map` | ZipControl 객체와 연결할 Map 객체 |

## ZipControlOption
**Type**: `Object`
ZipControl 생성 옵션 타입

**Properties**

| Name | Type | Default | Description |
| --- | --- | --- | --- |
| file | `HTMLInputElement` |  | zip 파일 업로드 input element |
| [encoding] | `string` | `&quot;&#x27;UTF-8&#x27;&quot;` | 인코딩 종류 |
| epsg | `string` |  | 좌표계 EPSG 코드 (숫자만, 예: '5179') |
| [getLayer] | `function` |  | 생성된 Layer 객체를 받을 콜백 함수 |
| [geometryType] | `Array.<(&#x27;point&#x27;\|&#x27;linestring&#x27;\|&#x27;polygon&#x27;)>` |  | 허용할 geometry 타입 배열 |
| [errorCallback] | `function` |  | 에러 발생 시 에러 메시지를 받을 콜백 함수 |
