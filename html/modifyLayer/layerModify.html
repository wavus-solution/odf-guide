원하는 수정 기능 버튼을 클릭 후 레이어를 수정합니다.

수정 완료 후 해당 기능 완료 저장 버튼 클릭 시 해당 결과값을 받을 수 있습니다.

1.(레이어 편집)버튼 클릭 후 지도에서 원하는 피처 선택 후 드래그하여 피처 수정

2.레이어에 피쳐 추가는 그리기도구를 이용하여 피쳐를 그린뒤 (레이어에 피쳐추가)버튼 클릭

3.(레이어 피처삭제)버튼 클릭 후 지도에서 원하는 피처 선택

4.(레이어피처이동)버튼 클릭 후 지도에서 원하는 피처 선택 후 마우스 드래그로 피처 이동

5.(레이어 피처 크기 조정 및 회전)버튼 클릭 후 피처 선택 후 마우스로 피처 크기 조정
